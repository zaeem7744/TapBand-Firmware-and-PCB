# WS2812_Test

Quick WS2812/NeoPixel test.

## Build
- Board: ESP32
- Library: Adafruit NeoPixel (if used in the sketch)
- Define LED pin and count for your setup; ensure a proper 5V rail and a level‑shifted data line if the run is long

## Notes
- Add a 330–470Ω series resistor on DIN and a large 5V bulk cap (>1000µF) for stability on long strips
